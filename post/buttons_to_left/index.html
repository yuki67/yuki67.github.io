Ubuntu 17.10で最大化/最小化ボタンを左へ移動する

2018-01-22     ubuntu  / メモ

これ↓

Ubuntu 17.10 のデフォルトだと右にあるが、これだと左端にあるドッグとの相性が悪いので変えた。

gsettings set org.gnome.desktop.wm.preferences button-layout 'close,maximize,minimize:'

でできた。

参考: Ubuntu 17.10 Tip: Move Window Buttons ‘min, max, close’ to Left

追記: gnome-tweak-tool からもできた。
